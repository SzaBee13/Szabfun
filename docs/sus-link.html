🔗 Sus Link Endpoints

Base URL: https://szb.pagekite.me/sus-link

Express.js

POST /create-custom-link

Request: { "url": "https://example.com" }

Response:

  • 200 OK – Link created! { "short": "https://szb.pagekite.me/abc123" }
  • 400 Bad Request – Missing URL 💀 { "error": "Missing required field: url" }

GET /get-custom

Request: ?id=abc123

Response:

  • 200 OK – Found it! { "url": "https://example.com" }
  • 404 Not Found – That ID ain't real 😢 { "error": "Custom link not found" }

GET /search

Request: ?query=example

Response:

  • 200 OK – Results found [ "https://example.com", "https://example2.com" ]
  • 400 Bad Request – Missing query param 💀 { "error": "Query parameter 'query' is required" }
