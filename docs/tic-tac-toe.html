🎮 Tic Tac Toe Endpoints

Base URL: https://szb.pagekite.me/tic-tac-toe

Socket.IO

POST /start

Request: { "playerX": "id1", "playerO": "id2" }

Response:

  • 200 OK – Game started! { "gameId": "abc123" }
  • 400 Bad Request – Missing player info 💀 { "error": "Both playerX and playerO fields are required" }

POST /move

Request: { "gameId": "abc123", "player": "id1", "pos": [0,1] }

Response:

  • 200 OK – Move accepted { "status": "ok", "board": [["X","", ""], ["", "O", ""], ["", "", ""]] }
  • 400 Bad Request – Invalid move 💀 { "error": "Invalid move: position already taken or out of range" }

GET /game-over

Request: ?gameId=abc123

Response:

  • 200 OK – Game over { "winner": "id1" }
  • 404 Not Found – No such game 😢 { "error": "Game not found" }
